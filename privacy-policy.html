Privacy Policy for Flycast for Android

Information collection and use

1. Crash data

In the event of an application crash or malfunction, and only if the "Automatically Report Crashes" option is enabled in the app settings, a crash report will be sent to Sentry.io. This report doesn't contain personally identifiable information and is only accessible to members of Flycast's development team. This information is only used to improve the quality of the app.
For more information about Sentry's privacy policy, visit https://sentry.io/privacy.

2. Online games

When emulating software which uses internet connectivity, Flycast will collect and transmit data as instructed by the game or software you are attempting to run. For example, when playing Phantasy Star Online, Flycast will transmit game data to the Phantasy Star Online game server. Flycast doesn't control which data the game sends to the server, but it is exclusively limited to data inside the emulator itself.

3. Third-party services

  • Downloading game box covers and metadata uses TheGameDB service. Using this feature will share with TheGamesDB your network's IP address as well as the game identifier and title of all games in your library. Please refer to TheGamesDB's Privacy Policy for details on how this information is processed. You can opt-out of this sharing by not using the "Fetch Box Art" feature in Flycast's settings.
  • By default, if you use the Discord application on the same computer as Flycast, the game you are currently playing will be shared with Discord. This is used to provide automatic presence status update. Please refer to Discord's Privacy Policy for details on how this information is processed. You can opt-out of this sharing by disabling the "Discord Presence" feature in Flycast's settings.
